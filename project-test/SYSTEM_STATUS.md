## 🛠️ 使用说明

### 1. 访问系统
```bash
# 前端界面
打开浏览器访问: http://127.0.0.1:3000

# API文档
访问: http://127.0.0.1:8000/docs
```

### 2. 查看实时数据
- 在Dashboard页面可以看到市场概览
- 在SatelliteModule页面查看ETF动量排名
- 在Settings页面配置策略参数

### 3. 生成交易建议
- 点击"生成决策"按钮
- 系统会基于当前市场数据生成建议
- 可导出为CSV格式的条件单
